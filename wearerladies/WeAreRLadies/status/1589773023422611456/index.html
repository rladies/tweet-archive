Time for some tips! Here are my most used RStudio shortcuts for Mac. If working on PC, swap cmd->ctrl and opt->alt

Pipe = cmd+shift+M
New R script = cmd+shift+N
New code chunk = cmd+opt+i
Run current chunk = cmd+shift+enter
