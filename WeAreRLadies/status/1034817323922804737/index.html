☝️ My first #rstats tip: use purrr::map_df() to read all .csv files in a 📂 and stick them in a single data frame:

f <- list.files(
"my_folder",
pattern = "*.csv",
full.names = TRUE)

d <- purrr::map_df(f, readr::read_csv, .id = "id")
